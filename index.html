Política de Privacidad

Esta aplicación no recopila ni almacena datos personales. Toda la información procesada se utiliza únicamente para publicar contenido automatizado en Facebook, sin intervención humana ni análisis individualizado.

El uso de esta aplicación está limitado a funciones automatizadas de publicación de contenido informativo. No se realiza seguimiento, perfilado ni almacenamiento de datos de usuarios.

Para más información o solicitudes de eliminación de contenido, puedes contactarnos directamente a través de la página de Facebook asociada.
